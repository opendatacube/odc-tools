Development (Conda)
-------------------

Create a conda environment and install this library in editable mode. This assumes
the [conda-forge](https://conda-forge.org/) channel is added.

```bash
conda create --name odc_tools datacube
conda activate odc_tools
cd libs/aio/
pip install -e .
```
